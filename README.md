# flood-segmentation
Flood area segmentation Project for ECE9309

### Dataset 

The data set used is from Kaggle https://www.kaggle.com/datasets/faizalkarim/flood-area-segmentation

### Approaches

1. Vision Transformers
2. Unet
3. ENet (Efficient Neural Network)
